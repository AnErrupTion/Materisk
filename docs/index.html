Welcome

Spaghetto is a simple, interpreted programming language not specialized in anything.

Spaghetto files uses the .spag extension, but you can also run code using the CLI. Also, Spaghetto is quite easy to learn.



Getting Started

To start using Spaghetto, you are gonna have to download the latest release or download the repository and build it yourself. The downloaded file will be in the zip format, extract to to where you want. Inside your new folder, there should be a spaghettoCLI.exe file. Run it to open the CLI interpreter (run code quickly in the console) or use --file to specify a file to run. If you want to run a file, but stay in the CLI, you can use the global run(path: String) method.

Value types

Spaghetto has so called Value types, these are built-in types that hold values. Technically, every variable ever is a value type. Currently, these Value types exist: Number, Function, Class, ClassInstance, List and String.
To create a string, you can simple use double quotes like this: "Hello World!" This code will create a string with the content Hello World!. Numbers are created by just, well, typing in numbers.

Comments

Spaghetto supports single-line comments. To create a comment, start the line with // and it will be ignored.

Variables

To define a variable, you use var myVar = <value>. This will define the variable myVar in the current "context" (function context, global context, etc.). To re-assign your variable, you can use myVar = 3 - this technically also works without having defined it previously, but it makes your code less readable if you use it like that.
A variable can contain any value type. Lists are used to store a list of values. These can also contain any value, in any order. To define a list, use var myVar = [1, 2, 3].

Semicolons

Semicolons aren't required if you don't want to use them. The only case where you need them is, for example, when using an if with an inline statement. Take a look at this example: if(myVar == 5) {; print("My var is 5"); }

Functions

Unlike in my previous, horrible, programming language, you can actually define proper functions with methods and return values. Use this example: func calcAvg(lst) {
 var sum = 0;

 for i = 0 while i < lst.size() step 1 {
  sum = sum + lst#i;
 }

 return sum / lst.size();
} This code will define a new function called calcAvg, which can be called by using calcAvg([3, 2, 6, 2, 4, 0]) or with whatever list you want to use.
